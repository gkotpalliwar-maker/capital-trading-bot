# Capital.com Trading Bot v2.3.0

ML Signal Scoring + Breakeven & Partial TP.

## Install
```bash
cd /root/trading-bot
git clone https://github.com/gkotpalliwar-maker/capital-trading-bot.git /tmp/v230
bash /tmp/v230/install.sh
rm -rf /tmp/v230
systemctl restart trading-bot
```

## New Commands
`/mlstats` `/retrain` `/mlthreshold` `/breakeven` `/partialtp` `/trademanage`

See `.env.example` for new variables.
